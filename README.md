# 홍길동 포트폴리오

풀스택 개발자 홍길동의 개인 포트폴리오 랜딩페이지입니다.

## 기술 스택

- HTML5
- CSS3 (순수 CSS, 외부 라이브러리 없음)
- Java Backend

## 주요 기능

- 히어로 섹션 (이름, 직함, CTA 버튼)
- 기술 스택 뱃지
- 프로젝트 카드 3개
- 이메일 · GitHub 연락처
- 다크 테마 기본 적용
- 모바일 반응형 레이아웃

## 실행 방법

`index.html` 파일을 브라우저에서 직접 열면 됩니다. 별도 서버나 빌드 과정이 필요 없습니다.

## 커스터마이징

- `style.css`의 `:root` 변수에서 색상 테마를 변경할 수 있습니다.
- `index.html`에서 이름, 프로젝트 내용, 연락처를 수정하세요.
